# Documentation

This directory contains the active project documentation.

- `roadmap.md`: verified implementation state and planned work
- `specs/`: current product and system behavior
- `guides/`: contributor and operator guidance
- `adr/`: major architectural decisions and why they were made

Historical or superseded material lives under `archive/docs/` and is not canonical.

- Specs index: `docs/specs/README.md`
- Guides index: `docs/guides/README.md`
- ADR index: `docs/adr/README.md`
